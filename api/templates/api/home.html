Games API Documentation

Endpoint Method Description
/api/game/
Query params (one required):
id: Game ID
name: Game name
GET Retrieve a single game by ID or name

Example: /api/game/?id=123 or /api/game/?name=Elden Ring
/api/games/
Query params (all optional):
filterBy(supports multiple filters):
- genre(Action,RPG)
- platform(windows,mac,linux)
- year(2021,2022)
sortBy(one of):
- metacriticScore
- price
- releaseDate
sortOrder(default: desc):
- asc
- desc
page: Page number
pageSize: Items per page (max: 100)
GET Get a paginated list of games with filtering and sorting

Example: /api/games/?filterBy=genre(Action,RPG)&platform(windows)&sortBy=metacriticScore
/api/games/recommend/
Query params (one required):
id: Reference game ID
name: Reference game name
GET Get 5 game recommendations based on a reference game:

Example: /api/game/recommend/?id=123 or /api/game/recommend/?name=Elden Ring
/api/games/create/
Request body:
name: string (required)
Optional fields:
release_date: YYYY-MM-DD
price: decimal
genres, tags, categories: comma-separated strings
developers, publishers: comma-separated strings
windows, mac, linux: boolean
... and all other fields from the model
POST Create a new game entry
/api/games/update/{id}/ PUT/PATCH Update an existing game
Supports partial updates with PATCH
/api/games/delete/{id}/ DELETE Delete a game

Example: /api/games/123/delete/
